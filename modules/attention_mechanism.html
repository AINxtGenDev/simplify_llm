Q-, K- und V-Vektoren im Transformer nachvollziehen

Was ist der Attention-Mechanismus?

Attention ist das Herzstück der Transformer-Modelle. Der Mechanismus erlaubt es, sich bei jedem Schritt auf genau die Token zu konzentrieren, die für das aktuelle Wort relevant sind – ähnlich wie wir Menschen beim Lesen bestimmte Wörter hervorheben.

Die drei Schlüssel-Vektoren: Query, Key und Value

  • Query (Q): „Wonach suche ich?“ – das aktuelle Token fordert Kontext an.
  • Key (K): „Was habe ich anzubieten?“ – beschreibt den Inhalt jedes Tokens.
  • Value (V): „Welche Information gebe ich weiter?“ – der Wert, der in die nächste Schicht fließt.

So funktioniert Attention:

1. Q-, K-, V-Vektoren erzeugen: Jedes Token wird mit gelernten Matrizen multipliziert.
2. Ähnlichkeit berechnen: Skalarprodukte zwischen Query und allen Keys bestimmen die Relevanz.
3. Skalieren & Softmax: Division durch √dk und Softmax sorgen für stabile Gewichte.
4. Gewichtete Summe: Values werden mit den Gewichten multipliziert und addiert → Kontextvektor.

Die Formel: Attention(Q, K, V) = softmax(Q·KT / √dk) · V

Warum ist das so mächtig?

Das Modell kann dynamisch erkennen, dass sich „Himmel“ auf „Farbe“ bezieht oder dass Pronomen wie „er“ auf ein bestimmtes Nomen zeigen. Hohe Attention-Scores markieren dabei den relevanten Kontext.
